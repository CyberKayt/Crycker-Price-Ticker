[ENG instruction]

Assembly Editor - This is to replace the properties (information) of the file
Functions - These are the functions of the build file
Build Stealer - This is for creating a build file with all the settings.

=====================================

~Assembly Editor~

Generate Assembly - Generates random embedded list
Random Assembly - Generates random words (large)
Clone data Application - Copies information from another file

Add properties for the build file - Activates adding properties to the file 
Change another app - Activates the change of file properties as for other file types.
Add icon for the build file - Activates an icon for the file

=====================================

~Functions~

Anti VM ( Virtual Machine ) - Activates protection against virtual machines.
Anti CIS - Activates protection against starting on RU systems.
Add AntiDump - Protection against process memory dumping.
Add String Encrypt - Protects strings of code with Gzip method
Output Console Application - Output file in console format
Output Winforms Application - Output file in a form ( hidden!)
Self Delete Build - Deletes the build file by itself
Active Logger - Activates the logging of the build file actions in the text file ( %AppData% )

Link - A link to the panel
Archive tag - Tag for the archive
Output file name - The name of the build file.
